<h2>ОПИСАНИЕ</h2>

<h3> DPAR - библеотека для чтение и записи данных из разных форматов файлов </h3>
<h3> список поддерживаемых форматов для анализа(будущая функция): </h3>

* JSON(СКОРО)

* TXT(СКОРО)

* Markdown(СКОРО)

* XML(СКОРО)
